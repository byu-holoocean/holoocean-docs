===========
ROS2 Bridge
===========

Seamlessly connect the HoloOcean simulator to the ROS 2 ecosystem with our official ROS 2 Bridge!

The ROS 2 Bridge enables publishing and subscribing to sensor data and control commands using standard ROS 2 topics,
making it easy to integrate HoloOcean into your robotics workflows. 
It includes support for autopilot control, simulation reset, and clock synchronization.

- Compatible with ROS 2 Humble  
- Launch with a single command
- Docker containers available for development and runtime  

Installation and usage instructions can be found on the GitHub repository:  
https://github.com/byu-holoocean/holoocean-ros
