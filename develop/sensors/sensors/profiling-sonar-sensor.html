Profiling Sonar

Holoocean has three different profiling sonar sensor implementations:

  • Octree Profiling Sonar

  • Raycast Profiling Sonar

  • GPU Profiling Sonar

These implementations have different tradeoffs in accuracy, speed, and memory usage.

Note

See Visualizing Profiling Sonar for examples of how to use this sensor.

Octree Profiling Sonar

Simulates a profiling sonar with octrees. It has a narrow elevation and wide asimuth, and projects the intensity onto the 2D time of return-azimuth plane.

See ProfilingSonar for the Python API.

Example sensor definition:

{
    "sensor_type": "ProfilingSonar",
    "socket": "SonarSocket",
    "Hz": 2,
    "configuration": {
        "Azimuth": 120,
        "Elevation": 1,
        "RangeMin": 1,
        "RangeMax": 60,
        "RangeBins": 512,
        "AzimuthBins": 512,
        "AddSigma": 0.15,
        "MultSigma": 0.2,
        "MultiPath": True,
        "ClusterSize": 5,
        "ScaleNoise": True,
        "AzimuthStreaks": -1,
        "RangeSigma": 0.1,
        "ShowWarning": True,
        "InitOctreeRange": 70,
        "ViewRegion": False,
        "ViewOctree": -10,
    }
}

Raycast Profiling Sonar

Simulates a profiling sonar with raycasting. It has a narrow elevation and wide asimuth, and projects the intensity onto the 2D time of return-azimuth plane.

See RaycastProfilingSonar for the Python API.

Example sensor definition:

{
    "sensor_type": "RaycastProfilingSonar",
    "socket": "SonarSocket",
    "Hz": 2,
    "configuration": {
        "RangeBins": 512,
        "AzimuthBins": 512,
        "RangeMin": 0.5,
        "RangeMax": 60,
        "Elevation": 1,
        "Azimuth": 120,
        "AddSigma": 0.0,
        "MultSigma": 0.0,
        "MultiPath": False,
        "ShowWarning": False,
        "ViewRegion": False,
        "IgnorePlants": False
    }

GPU Profiling Sonar

Simulates a profiling sonar with GPU rendering. It has a narrow elevation and wide asimuth, and projects the intensity onto the 2D time of return-azimuth plane. It utilizes the GPU rendering pipeline, for faster simulation speed.

See GPUProfilingSonar for the Python API.

Example sensor definition:

{
    "sensor_type": "GPUProfilingSonar",
    "socket": "SonarSocket",
    "Hz": 2,
    "configuration": {
        "RangeBins": 512,
        "AzimuthBins": 512,
        "RangeMin": 0.5,
        "RangeMax": 60,
        "Elevation": 1,
        "Azimuth": 120,
        "AddSigma": 0.0,
        "MultSigma": 0.0,
        "MultiPath": False,
        "ShowWarning": False,
        "ViewRegion": False,
        "IgnorePlants": False
    }
}

Resolution Calculation for Raycast and GPU Profiling Sonars

Resolution for the raycast and GPU profiling sonars can be calculated using the same equations as the imaging sonars. Please refer to Resolution Calculation for Raycast and GPU Imaging Sonars for more information.

Note

The values for Azimuth, Elevation, RangeMax, RangeMin, and RangeRes in the equations should be set to the values used in your sensor configuration.
